# shree Jewellerys

A modern and responsive Jewellery E-Commerce Website developed using React, TypeScript and Vite.

## Features

- Responsive Design
- Home Page
- About Us
- Contact Us
- Product Categories
- Product Details
- Shopping Cart
- Search Products
- WhatsApp Order Integration

## Technologies Used

- React
- TypeScript
- Vite
- CSS

## Run the Project

```bash
npm install
npm run dev
```
